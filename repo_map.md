# VendorIQ Repo Map for AI Agents

Start here:

1. `SKILL.md` — canonical execution instructions.
2. `README.md` — full user-facing documentation.
3. `skills.md` — formal skill definition.
4. `agents/agent_registry.md` — required agent responsibilities.
5. `guardrails/` — validation rules.
6. `playbooks/` — execution playbooks.
7. `templates/exhaustive_final_report_template.md` — final report structure.
8. `src/universal_vendor_orchestrator.py` — optional local CLI orchestrator.

For online execution:
- Do not run code.
- Treat the repo as an instruction package.
- Use browsing/search.
- Return `exhaustive_final_report.md`.

For local execution:
- Install `requirements.txt`.
- Configure `.env`.
- Run `src/universal_vendor_orchestrator.py`.

Ignore during first-pass execution:
- `.vscode/`
- `artifacts/`
- `.gitignore`
- contribution-only files
